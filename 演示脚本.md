# 灵析 (Lingxi) — 智能体演示视频脚本

> **严格控制在 7 分钟以内**
> 录制前准备：确保后端服务已启动（`python app.py`），浏览器已打开 `http://localhost:5000`
> 建议提前跑一遍完整流程，确认所有 AI 接口响应正常

---

## 第一幕：开场与登录（约 40 秒）

### 操作
1. 页面加载完成后，**停留 2 秒**展示 Hero 区域（品牌名 + 标语 + 右侧光球动效）
2. 鼠标**缓慢向下滚动**，展示六大特色功能卡片（多智能体协同 / 智能学习路径 / 能力画像追踪 / 文档沉浸学习 / 智能测验系统 / 个性化推荐）
3. 点击 **"立即开始"** → 弹出登录卡片
4. 输入用户名密码 → 点击 **"登录"** → 等待过渡动画

### 台词

> 大家好，这是**灵析**——一款基于多智能体协同的个性化学习系统。
> 它的核心亮点是：三个 AI 智能体组成团队，以**流水线协作**的方式为你完成知识分析、资源生成和学习路径规划。
> 我现在登录系统，给大家实际演示。

---

## 第二幕：全体协同 — 三 Agent 流水线协作（约 2 分 30 秒）★ 核心重头戏

### 操作
1. 进入主界面后**暂停 2 秒**，展示三栏布局（侧边栏 + 聊天区 + 右侧画像面板）
2. 快速点击侧边栏用户菜单 → **"课程中心"**，停留 3 秒展示课程看板，然后点返回
3. 在左侧边栏点击 **"全体协同"** 智能体卡片
4. 在输入框输入：**"我想学习快速排序算法"**
5. 点击发送，**等待 AI 响应（约 10-15 秒）**

> 等待期间可以解说（见下方台词）

6. 响应完成后，**缓慢向上滚动**，依次展示：

### 展示顺序与台词配合

**（1）协同链路可视化**

> 响应完成了。大家看最上面这条**协同链路**：你 → 学习分析师 → 资源生成师 → 路径规划师 → 完成。还标注了耗时和参与 Agent 数量。

**（2）学习分析师卡片（蓝色）**

> 第一个 Agent 是**学习分析师**。它基于讯飞星火大模型，对"快速排序"这个主题做了完整诊断——包括知识难度评估、关键词提取、薄弱点分析和学习风格建议。它还会参考你在系统里的做题数据来给出个性化建议。

**（3）路径规划师卡片（橙色）**

> 第三个 Agent 是**路径规划师**。它接收了前两个 Agent 的分析结果和资源摘要，为你生成了一条**分步骤的学习路径**——从基础概念到综合实践，每一步都有明确的行动指引和预估时间。这就是"因材施教"。

**（4）资源生成师卡片（绿色，放最后）— 重点展示多模态资源**

> 最后是**资源生成师**，这是多模态资源生成的核心。
>
> 首先，它生成了**知识卡片**——可以看到这些紫色卡片，点击可以**翻转**查看答案，分为基础、中级、高级三个难度。
>
> 然后是**代码案例**——直接给出了可运行的快速排序代码，附有语言标签和详细解释。
>
> 最关键的是——系统**自动调用了 B 站搜索 API**，根据分析师提取的关键词，实时抓取了相关的优质教学视频。你看这里，直接嵌入了 B 站视频链接，点击就能跳转播放。学习资料不用自己找，AI 帮你一站式搞定。

---

## 第三幕：AI 解题群 — 三模型并行对比（约 1 分钟）

### 操作
1. 在左侧边栏点击 **"AI 解题群"** 切换到群聊模式
2. 输入：**"解释一下什么是时间复杂度，O(1) 和 O(n log n) 有什么区别？"**
3. 等待三个模型同时回复（约 5-10 秒）
4. 展示三个回答的对比效果

### 台词

> 接下来看另一个智能体交互模式——**AI 解题群**。
> 我问一个问题，**三个不同的 AI 模型同时回答**：讯飞星火、混元Turbo、Qwen3。
> 每个模型有不同的训练背景和表达风格。你可以对比三个答案，获得更全面的理解。就像你在学习群里提问，三个学霸同时给你不同的解题思路。

---

## 第四幕：多模态能力 — 图片识别（约 1 分钟）

### 操作
1. 切回普通聊天模式
2. 点击输入框上方的 **图片上传按钮**（相机图标）
3. 选择一张**包含文字/公式/图表的图片**（提前准备好，比如一张数学公式截图或算法图）
4. 图片上传后在输入框上方展示缩略图预览
5. 点击发送，等待 AI 识别并回复（约 5-10 秒）

### 台词

> 灵析还支持**多模态图片理解**。
> 我上传一张包含数学公式的图片。
> 系统调用了讯飞星火的**视觉理解接口**，通过 WebSocket 协议传入 base64 编码的图片数据，AI 识别出图片中的公式和文字内容，并给出详细的推导和解答。
> 这意味着你可以直接拍题目、截图，让 AI 来帮你分析。

---

## 第五幕：快捷工具栏 — 一键脑图 + AI 翻译（约 50 秒）

### 操作
1. 展示输入框上方的工具栏（鼠标从左到右缓慢划过）
2. **实操一：一键脑图（重点）**
   - 点击 **"一键脑图"** → 弹出脑图窗口
   - 输入主题：**"快速排序"**
   - 点击 **"生成脑图"**
   - 等待 AI 生成（约 5-8 秒），展示渲染完成的树形思维导图
   - 用鼠标**缓慢滚动**导图，展示分支结构
3. **实操二：AI 翻译（快速展示）**
   - 点击 **"AI 翻译"** → 弹出翻译窗口
   - 输入一段中文，选择翻译为英语
   - 展示翻译结果即可，不用详细解说

### 台词

> 聊天框上方有一排快捷工具栏，包括文档诊断、一键脑图、代码纠错、出题助手、AI 翻译、图片上传。
>
> 我重点演示**一键脑图**。输入主题"快速排序"，点击生成。
>
> 系统调用讯飞星火大模型生成 Markdown 结构化大纲，然后前端实时解析渲染成**可视化思维导图**。大家看这个树形结构——根节点展开为多个分支，每个分支下有叶子节点。
>
> 生成过程中还有一个智能过滤机制：系统会自动屏蔽"参考资料"、"外部链接"等不属于知识主干的内容，只保留高含金量的核心知识点。
>
> 另外还有 AI 翻译，支持八种语言互译；出题助手可以自动生成选择题并收录错题到错题本。时间关系就不一一展开了。

---

## 第六幕：AI 能力画像系统（约 50 秒）

### 操作
1. 确保当前在聊天页面，右侧面板可见
2. 如果右侧面板未显示能力画像，点击右侧面板的**"能力画像"** 标签/区域
3. **缓慢展示**六维雷达图（知识掌握 / 认知能力 / 纠错能力 / 兴趣活跃 / 目标完成 / 学习习惯）
4. 向下滚动展示**薄弱知识点**列表和**兴趣标签**
5. 切到**"我的课程"**标签，展示学习统计数据（学习天数 / 资源数 / 累计时长 / 收藏数）
6. 切到**"错题集"**标签，展示错题卡片（如有）

### 台词

> 接下来看**AI 能力画像系统**，这是灵析实现个性化学习的数据基础。
>
> 右侧这个六维雷达图涵盖了知识掌握、认知能力、纠错能力、兴趣活跃、目标完成、学习习惯六个维度。
>
> 系统通过两个渠道来构建画像：一是**AI 自动提取**——后端调用讯飞星火大模型对每次对话内容进行分析，自动识别你的兴趣方向和知识薄弱点，写入用户画像数据；二是**行为追踪**——记录你的做题正确率、学习频次、课程完成度等数据。
>
> 画像数据会实时同步到服务端，换设备登录也不会丢失。
>
> 下面还能看到薄弱知识点列表和兴趣标签，帮助精准定位需要加强的方向。
>
> 这一块的统计数据——学习天数、资源数、累计时长——都是和用户画像数据实时挂钩的。

## 第七幕：结尾（约 20 秒）

### 操作
回到聊天主界面，让镜头**定格在完整的系统界面**。

### 台词

> 最后总结一下灵析的核心技术亮点：
>
> **第一，多智能体协同架构**——学习分析师、资源生成师、路径规划师三个 Agent 串行流水线协作，后序 Agent 充分利用前序 Agent 的分析结果，实现真正的上下文感知。
>
> **第二，AI 驱动的个性化能力画像**——基于大模型对话分析 + 行为数据追踪，自动构建六维能力雷达图，实现数据驱动的因材施教。
>
> **第三，多模态资源自动生成**——知识卡片、代码案例、B 站视频实时抓取，学习资源一站式产出。
>
> **第四，多模型并行 + 视觉理解**——讯飞星火、混元Turbo、Qwen3 三模型同时回答，配合图片上传识别，实现真正的多模态交互。
>
> 谢谢大家。
