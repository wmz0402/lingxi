### 二级课程1：微积分一（Calculus I）
#### 第1章 极限与连续性
##### 一、单选题（4道）
1. $f(x)=\dfrac{x^2-4}{x-2}$ 在 $x=2$ 处极限值为（）
A. 无定义 B. 4 C. 0 D. 2
**答案：B**
**解析：因式分解约去$x-2$，化简为$x+2$，代入得4**
2. $x\to0$ 时等价无穷小错误的是（）
A. $\sin x \sim x$ B. $\ln(1+x)\sim x$ C. $e^x-1\sim x$ D. $\dfrac1x \sim x$
**答案：D**
3. 下列函数在全体实数域连续的是（）
A. $y=\dfrac1x$ B. $y=|x|$ C. $y=\tan x$ D. 分段$x>0,y=1;x\le0,y=2$
**答案：B**
4. $\lim\limits_{x\to\infty}\dfrac{3x^2+2}{x^2-1}=$（）
A. 3 B. 0 C. ∞ D. -1
**答案：A**

##### 二、多选题（2道）
1. 函数在$x_0$连续的充要条件包含（）
A. $f(x_0)$有定义 B. $\lim\limits_{x\to x_0}f(x)$存在 C. 极限值=$f(x_0)$ D. $f(x)$在$x_0$可导
**答案：ABC**
2. 间断点分类包含（）
A. 可去间断点 B. 跳跃间断点 C. 无穷间断点 D. 光滑间断点
**答案：ABC**

##### 三、填空题（2道）
1. $\lim\limits_{x\to0}\dfrac{\sin5x}{3x}=$ ________
**答案：$\dfrac53$**
2. $\lim\limits_{x\to0}(1+2x)^{\frac1x}=$ ________
**答案：$e^2$**

##### 四、计算题（1道）
1. 求极限：$\lim\limits_{x\to1}\dfrac{\ln x}{x-1}$
**解：等价无穷小$\ln x=\ln(1+x-1)\sim x-1$，原式$=\lim\limits_{x\to1}\dfrac{x-1}{x-1}=1$**

##### 五、证明题（1道）
1. 证明：若$f(x)$在$x_0$处可导，则$f(x)$在$x_0$一定连续。
**证明：**
由导数定义：$f'(x_0)=\lim\limits_{\Delta x\to0}\dfrac{f(x_0+\Delta x)-f(x_0)}{\Delta x}$
变形$\Delta y = f'(x_0)\cdot\Delta x + o(\Delta x)$，当$\Delta x\to0$，$\Delta y\to0$，满足连续定义。

// C栈/堆内存对比演示
#include <stdio.h>
#include <stdlib.h>
void test() {
    int stack_var = 10; // 栈
    int *heap_var = (int*)malloc(sizeof(int)); // 堆
