# 灵析 (Lingxi) — AI 智能个性化学习辅助系统

灵析（Lingxi）是一款基于科大讯飞星火大模型（Spark API）与多模态视觉理解技术构建的高颜值、全功能、智能个性化学习辅助平台。系统旨在通过 AI 导师的高效协同，为用户提供集**路径规划、文档诊断、一键脑图、代码纠错、自主测试、多模态提问与能力画像**于一体的全新沉浸式学习体验。

---

## 🌟 核心功能特性

### 1. 🤖 多智能体协同与个性化路径规划
- **学习规划师**：只需输入您想学习的任意主题，AI 就会为您量身制定结构化、循序渐进的系统化学习路线图。
- **资源生成师**：智能检索系统内置章节、模拟题库，并自动关联拉取相关的 Bilibili 优质视频教学资源，告别信息检索焦虑。
- **多轮会话记忆**：大模型全面接入多轮历史上下文记忆，学习过程连贯，提问体验丝滑。

### 2. 📊 全方位的个性化能力画像
- **多维雷达图**：系统通过对用户聊天主题、测试表现及行为频率的深度追踪，生成包含“知识掌握”、“认知能力”、“纠错能力”、“兴趣活跃”、“目标完成”以及“学习习惯”的六维雷达能力画像。
- **活跃追踪**：对用户的文档诊断、测试答案提交、代码纠错及脑图生成等行为进行毫秒级的动态日志记录与时间降序排版，随时掌握学习足迹。

### 3. 📝 自主出题测试与错题本系统
- **一键出题**：用户可自定义出题的主题、题量和难度级别（初级/中级/高级），AI 自动生成符合规范的单项选择题试卷。
- **在线评测**：提供交互式做题界面，支持在线提交，自动判断对错并给出详尽的解析。
- **智能错题本**：自动收录错题，详细记录错误原因并归纳整理，支持随时二次重做与清空。

### 4. 🧠 一键思维脑图生成
- **自适应课程脑图**：内置多学科（高等数学、线性代数、概率论、数据结构与算法、计算机体系结构、操作系统、编译原理、Java核心编程、Python进阶、MySQL数据库等）精品课程，其章节底部的思维导图全面重构，支持自动将 Markdown/HTML 格式 of 课程正文，动态解析为多色相间的精美多层级树状思维导图。
- **交互生成**：聊天过程中，AI 的任何知识性回复均支持一键生成对应的脑图进行直观阅读与保存。

### 5. 💻 专业级代码一键纠错优化
- 支持一键上传 `.cpp`、`.py`、`.java`、`.js`、`.html` 等多语言源码文件。
- AI 导师自动完成深度语法检测、逻辑分析与内存泄漏排查，并提供左右分栏对比的修改后代码，支持一键复制代码。

### 6. 📂 文档深度诊断与分析
- 支持批量上传 PDF、Word（`.doc`/`.docx`）、Markdown、Excel（`.xlsx`/`.xls`/`.csv`）、文本（`.txt`）等多格式学习资料。
- 自动提取文本内容，并针对性地对文档结构、难点解法及提问要求给出深度诊断。

### 7. 📸 多模态图片理解与截图粘贴
- 深度整合科大讯飞 Spark 图像理解 websocket 接口。
- 支持在**主对话框**与**课程学习对话框**中一键上传本地图片，或者**直接在对话框中 Ctrl+V 粘贴屏幕截图**。
- 大模型可直接识别图像中的公式、图表、手写题目及文字，并给出严谨学术的推导与解答。

## 🛠️ 技术栈与架构设计

### 前端
- **核心逻辑**：原生 JavaScript、HTML5。
- **视觉设计**：极简现代美学 CSS 系统（毛玻璃质感、微交互渐变动效、响应式双栏布局）。
- **数学排版**：全面接入 MathJax 3.0，支持行内（`$...$`）与行间（`$$...$$`）LaTeX 数学公式的高清无缝渲染。

### 后端
- **核心框架**：Python + Flask (轻量高效)。
- **持久化**：SQLite3 轻量级关系型数据库，实现多用户数据隔离与本地同步。
- **AI 核心驱动**：`spark-ai-python` 官方 SDK（大模型推理） + `websocket-client`（多模态图像流传输）。

## 🚀 快速启动指南

### 1. 克隆并安装依赖
在项目根目录下，使用 `.venv` 虚拟环境安装所需依赖：
```bash
# 激活虚拟环境（Windows 示例）
.venv\Scripts\activate

# 安装必要依赖包
pip install flask flask-cors sqlite3 websocket-client spark-ai-python
```

### 2. 配置密钥证书
打开 `app.py`，配置您的科大讯飞 Spark 开发者凭证：
```python
# app.py 配置区
SPARK_APP_ID = "您的APPID"
SPARK_API_KEY = "您的APIKEY"
SPARK_API_SECRET = "您的APISECRET"
```

### 3. 运行本地服务器
```bash
python app.py
```
启动成功后，浏览器访问 `http://127.0.0.1:5000` 即可开始使用。
