# 灵析 (Lingxi) — AI 智能个性化学习辅助系统

灵析（Lingxi）是一款基于科大讯飞星火大模型（Spark API）与多模态视觉理解技术构建的高颜值、全功能、智能个性化学习辅助平台。系统旨在通过 AI 导师的高效协同，为用户提供集路径规划、文档诊断、一键脑图、代码纠错、自主测试、多模态提问与能力画像于一体的全新沉浸式学习体验。

线上免部署演示版访问地址：https://lingxi-jhyc.vercel.app/

---

## 核心功能特性

### 1. 多智能体协同与个性化路径规划
* 学习规划师：只需输入您想学习的任意主题，AI 就会为您量身制定结构化、循序渐进的系统化学习路线图。
* 资源生成师：智能检索系统内置章节、模拟题库，并自动关联拉取相关的 Bilibili 优质视频教学资源，告别信息检索焦虑。
* 多轮会话记忆：大模型全面接入多轮历史上下文记忆，学习过程连贯，提问体验丝滑。

### 2. 全方位的个性化能力画像
* 多维雷达图：系统通过对用户聊天主题、测试表现及行为频率的深度追踪，生成包含“知识掌握”、“认知能力”、“纠错能力”、“兴趣活跃”、“目标完成”以及“学习习惯”的六维雷达能力画像。
* 物理联动学习时长：日均学习时长与右侧学习统计卡片（学习总时长与累计天数）物理对齐，计算公式实时挂钩，绝无数据冲突。
* 标签自愈清洗：若因历史测试做错题目而在薄弱知识点中混入了长句题干残余，系统在登录与刷新时会自动运行自愈净化算法，自动剔除长度大于12字符的非法主题，保持界面整洁精致。

### 3. 自主出题测试与错题本系统
* 一键出题：用户可自定义出题的主题、题量和难度级别（初级/中级/高级），AI 自动生成符合规范的单项选择题试卷。
* 智能错题本：自动收录错题，支持查看解析和随时二次重做。错题卡片完全应用 Markdown 编译与 MathJax 数学公式重绘，并且在选项上做了 HTML 特殊字符转义（如小于号等），彻底避免了文本截断与公式乱码。

### 4. 动态思维脑图生成
* 自适应课程脑图：内置多学科精品课程，章节底部的思维导图已全面重构。
* 智能杂音过滤：生成脑图时，系统会自动过滤并屏蔽长文本、表格源码、代码块，以及“参考资料”和外部网址链接，只留下高含金量的核心知识大纲。
* 零内容分支隐藏：当某些章节标题下无实际可展示的内容时，系统会自动隐藏空分类，消除空白方框。

### 5. 专业级代码纠错与在线运行
* 支持一键上传 .cpp、.py、.java、.js、.html 等多语言源码文件。
* AI 导师自动完成深度语法检测、逻辑分析与内存泄漏排查，并提供左右分栏对比的修改后代码。
* 深度接入 JDoodle 编译运行接口，在代码纠错比对页支持一键编译、输入参数并在线查看代码运行结果。

### 6. 文档深度诊断与分析
* 支持批量上传 PDF、Word、Markdown、Excel、文本等多格式学习资料。
* 自动提取文本内容，并针对性地对文档结构、难点解法及提问要求给出深度诊断。

### 7. 多模态图片理解与截图粘贴
* 深度整合科大讯飞 Spark 图像理解 websocket 接口。
* 支持在主对话框与课程学习对话框中一键上传本地图片，或者直接在对话框中 Ctrl+V 粘贴屏幕截图，大模型可直接识别图像中的公式、图表、手写题目及文字，并给出严谨学术的推导与解答。

## 新手配置与快速启动指南

以下是面向新手的零基础快速部署与运行指导：

### 第一步：获取开发者凭证密钥

1. 科大讯飞星火大模型凭证：
   * 访问讯飞开放平台官网注册账号。
   * 控制台创建一个应用，并在大模型服务页面获取您的 APPID、APIKEY 和 APISECRET。
   * 本系统同时使用了文本对话接口和多模态图像理解接口。

2. JDoodle 编译接口凭证（若需要在线运行代码）：
   * 访问 JDoodle 官网并注册开发者账号。
   * 在控制台获取您的 Client ID 和 Client Secret。每次在线编译代码会消耗相应的额度。

### 第二步：配置本地运行环境

项目依赖 Python 3.8 及以上版本。请在项目根目录下配置虚拟环境并安装相关依赖包。

使用命令行（如 PowerShell 或 CMD）执行：
```powershell
# 1. 确保在项目根目录下，创建虚拟环境（若未创建）
python -m venv .venv

# 2. 激活虚拟环境
# Windows 系统：
.venv\Scripts\activate
# macOS/Linux 系统：
source .venv/bin/activate

# 3. 安装后端所需的必要 Python 库
pip install flask flask-cors websocket-client spark-ai-python
```

### 第三步：配置后端应用密钥

打开项目根目录下的 app.py，定位到文件顶部的配置区域，将您第一步获取到的凭证填写进去：

```python
# app.py 约第 30-45 行
SPARK_APP_ID = "填写您的讯飞APPID"
SPARK_API_KEY = "填写您的讯飞APIKEY"
SPARK_API_SECRET = "填写您的讯飞APISECRET"

# JDoodle 配置，用于支持在线代码编译与运行
JDOODLE_CLIENT_ID = "填写您的JDoodle Client ID"
JDOODLE_CLIENT_SECRET = "填写您的JDoodle Client Secret"
```

### 第四步：初始化本地数据库与数据同步

系统在首次运行前，必须先初始化 SQLite3 数据库，以便能够正确保存用户的注册状态、学习进度以及错题集。

在激活了虚拟环境的终端中，顺次运行以下脚本：
```bash
# 1. 初始化数据库结构（建立表并注入课程初始资源）
python init_db.py

# 2. 同步本地 Markdown 章节教材文本到数据库中
python sync_md_data.py
```
提示：如果未来您修改了本地的 .md 教材文档（如 编程语言.md 等），可以随时重新运行 python sync_md_data.py 来刷新数据库中的课程内容。

### 第五步：运行与本地访问

在终端运行以下指令启动 Flask 后端服务器：
```bash
python app.py
```
启动成功后，控制台会输出本地监听地址。在浏览器中打开以下链接即可体验完整功能：
```text
http://127.0.0.1:5000
```

## 部署说明

### 本地部署
直接按照“快速启动指南”中的步骤配置 Python 依赖，使用 python app.py 并在本地浏览器中访问即可。

### 线上部署 (如 Vercel 部署)
本系统支持与 Vercel 云平台深度结合，通过配置根目录下的 vercel.json，可以实现将 Python Flask 后端无缝转换为 Vercel Serverless Functions 运行，前端静态资源自动 CDN 加速分发，免去了管理云服务器的繁琐。

当前已与 Vercel 结合并开通访问的线上稳定演示地址为：https://lingxi-jhyc.vercel.app/

部署您自己的服务步骤如下：
1. 将本项目推送至您的 GitHub 私有或公开仓库。
2. 登录 Vercel 控制台，点击 Add New Project。
3. 选择您推送的仓库并导入，系统会自动识别 vercel.json 并配置好后端云函数路由。
4. 请在 Vercel 网页控制台的 Settings - Environment Variables 中，将 app.py 里配置的星火密钥及 JDoodle 密钥配置为对应的环境变量，以确保云端访问安全。

## 常见排错与 FAQ

1. 为什么页面显示了 LaTeX 公式源码，没有渲染成数学公式？
   * 错题集或章节内容动态生成时，MathJax 需要重新排版。本系统已在每次渲染更新 DOM 后自动调用 window.MathJax.typesetPromise() 进行重新重绘编译。请确保网络环境正常，能够成功从 CDN 加载 MathJax 的 JS 库。

2. 为什么点击历史对话，右侧页面还是欢迎词没有变化？
   * 此前在登录自动加载时，空欢迎词页面可能会覆盖本地 localStorage 中的历史，这在最近的重构中已经通过handleLogin自动保护逻辑修复。如果先前产生的历史对话数据已被永久覆盖污染，建议直接点击侧边栏的“清空历史”按钮，重新对话即可完美还原。

3. 上传代码并点击在线运行，控制台报错或无输出？
   * 请检查 app.py 顶部的 JDOODLE_CLIENT_ID 和 JDOODLE_CLIENT_SECRET 是否已配置。如果配置正确，请登录 JDoodle 查看您的每日编译调用额度是否已经用尽。
